# ImageHosterApp
developing an Image Hoster project from scratch which store the data in the PostgreSQL database.
# Function performed
1.Upload an Image<br/>
2.User Registration<br/>
3.User Login<br/>
4.Upload Image<br/>
5.Display an Image<br/>
6.Edit an Image<br/>
7.Delete an Image<br/>
8.Tags<br/>
